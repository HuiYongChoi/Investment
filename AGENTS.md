Investment Navigator

Investment Navigator는 투자자의 건강한 가치투자 판단을 돕기 위해 **DART(전자공시시스템)**와 **KRX Open API(한국거래소)** 데이터를 결합하여 실시간 데이터를 동기화하고 기업 실적과 주가 정보 및 차트 데이터를
한눈에 정리하여 즉각적으로 제공해주는 어플리케이션입니다.

투자 관련 기능은 아래와 같습니다.


## A. 투자관련 기능

1. 기업관련 정보
1-1. 검색된 기업의 당일 시가와 종가 그리고 현재가에 비율을 함께 표기합니다.
1-2. 가시성이 확보되며, 마우스를 올렸을 때 가격과 시가, 종가, 등락률을 표기하며, 일봉/주봉/연봉 등을 선택하여 볼 수 있게합니다.
1-2. 분석하고자 한국 상장 기업의 최근 3년간 사업보고서 및 분기별 보고서를 즉각 찾아 리스트업합니다.

2. 재무제표 분석
2-1. 수익성, 성장성, 효율성 세가지 측면으로 분류하여 최근 3년간 기업의 연도별 분기별 재무제표를 3차원 종합 평가합니다.
2-1-1. 수익성: 영업이익률, EPS성장률, 순이익성장률 등을 당기, TTM, 분기별로 전년도/전분기 대비 등락률을 표시합니다.
2-1-2. 건전성: 자본증감률, 부채증감률, 부채비율, 현금보유량, 이자보상비율 등을 당기, TTM, 분기별로 전년도/전분기 대비 등락률을 표시합니다. 
2-1-3. 효율성: ROE, ROA, ROIC, 매출채권회전률, 재고회전률 등을 당기, TTM, 분기별로 전년도/전분기 대비 등락률을 표시합니다. 

3. 기술적 분석
3-1. 기술적 분석을 선택할 수 있게 하는 창을 표기합니다. (스토캐스틱, RSI, MACD, 골든크로스/데드크로스, 볼린저 밴드 등)
3-2. 차트 아래 기술적분석과 관련된 차트를 추가적으로 표기하며 차트에는 매수/매도 표기를 함께 합니다.
3-3. 최종적으로 기술적분석에 따른 매수, 매도, 중립에 대한 의견이 각각 몇개인지를 취합하여 표기합니다.


추가적으로 **Kakao 로그인**을 지원하며 **Gemini AI**를 통한 투자 브리핑 기능을 제공합니다.


## B. 프로그램의 기술적 명세

4. 투자정보 관련 API
투자정보 실시간 연동을 위해 아래 API를 활용합니다. (* 관련정보는 폴더 내 파일 AWS.txt에 있습니다.)

4-2. 📊 데이터 파이프라인 최적화 (DART, KRX)

4-2-1. **DART API (기업의 내재 가치 파악)**
**역할**: 재무제표(당기순이익, 자산, 부채, 자본 등) 등 펀더멘털 데이터 추출
**활용**: `수익성(영업이익률, EPS 성장률)`, `효율성(ROE, ROA)`, `건전성(부채비율, 이자보상비율)`의 3차원 종합 평가 자동 계산

4-2-2. **KRX Open API (시장의 실시간 흐름 파악)** 
**역할**: 당일 주가, 거래량 실시간 시세 조회 및 OHLCV(고가, 저가, 시가, 종가) 차트 일봉 데이터 제공.
**활용**: 반응형 캔들 차트(MA5, MA20 오버레이) 드로잉, RSI, MACD, 스토캐스틱, 볼린저 밴드 등 기술적 지표 완전 자동 계산

4-2-3. Google AI Studio API (AI 투자 브리핑 및 리포트 자동 생성)
**Gemini 2.0 Flash 통합**: 수집된 재무 데이터 및 평가 점수를 바탕으로 "가치 투자 원칙 점수"를 계산하고, 핵심만을 짚은 요약 리포트를 한국어로 생성합니다.
**PDF 전문가 리포트**: 복잡한 인쇄 과정 없이 화면 내의 대시보드와 AI 브리핑을 고해상도 PDF 리포트로 바로 내보낼 수 있습니다. (`html2pdf` 라이브러리 사용)


5. 편리한 인증 및 개인화 (* 카카오 API 관련 정보는 폴더 내 파일 AWS.txt에 있습니다.)
**카카오 로그인**: Kakao JS SDK를 이용해 원클릭으로 간편하게 로그인할 수 있습니다.

5-1. 투자 브리핑을 카카오톡에서 나에게 즉각 보낼 수 있는 기능을 가집니다.
